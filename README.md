fun-robo
========
Play with robot

## Valid Commands 
  * PLACE x,y,face 
  * MOVE 
  * LEFT 
  * RIGHT 
  * REPORT 

## How to run FunRobo
  * commands from command line parameters
        ruby robo.rb 'PLACE 2,2,NORTH' MOVE LEFT MOVE RIGHT  REPORT 'PLAVE 4,3,SOUTH' MOVE REPORT
  * commands from file 
       ruby robo.rb filename
